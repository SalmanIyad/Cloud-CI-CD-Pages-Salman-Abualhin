What is CI/CD?

CI/CD stands for Continuous Integration and Continuous Deployment/Delivery.
It is a method to frequently deliver apps to customers by introducing automation into the stages of app development.
CI automatically tests and builds code when a developer pushes changes.
CD automatically deploys the tested code to production. This process reduces manual errors and speeds up software updates.

My Pipeline

When I push my code changes to the main branch on GitHub, a GitHub Actions workflow is automatically triggered.
It checks if all required files exist, builds an artifact of my website, and then deploys it directly to GitHub Pages.
